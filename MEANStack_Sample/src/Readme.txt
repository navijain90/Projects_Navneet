1. Install MongoDB, Nodejs,Express
2. Download the angularjs file, Bootstrap.
3. Nodejs will give you npm.
4. npm install -g express --save (-g ==global)
5. npm install -g nodemon --save (This is to automatic restart the server when there is a change in the code)
6. mongodb requires data/dir in the C drive. This is where all the data will be stored.
7. to use mongo, start the server mongod first and then mongo
8. db.meows.find({text:/hi/})  Find the document with Hi
